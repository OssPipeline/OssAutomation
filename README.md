# OssAutomation
Jenkins와 Kubernetes를 활용한 CI/CD 환경 구축 및 자동화

## 💙Commit Convention
[Feat] 새로운 기능 추가 \
[Fix] 버그 해결 \
[Design] UI 디자인 변경 (css 등) \
[Style] 코드 수정 없음 (코드 형식, 세미콜론 추가 등) \
[Chore] 빌드 업무, 패키지 매니징, gitignore \
[Build] 빌드 관련 파일 수정 \
[Docs] 문서 (README.md 등 문서 추가, 수정, 삭제) \
[Refactor] 코드 리팩토링 \
[Test] 테스트 (테스트 코드 추가)
